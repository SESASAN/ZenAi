# ZenAI

ZenAI es una interfaz web de chat diseñada para interactuar con modelos de Inteligencia Artificial de manera simple, moderna y escalable. 

Este proyecto forma parte del Proyecto Final del diplomado en Desarrollo Fullstack y será desarrollado de manera incremental:

- Fase 1: Maquetación base con HTML y CSS.
- Fase 2: Lógica e interactividad con JavaScript.
- Fase 3: Componentización con React, integración con Firebase y despliegue.

## Descripción del Proyecto

ZenAI es una aplicación web tipo chat que permite a los usuarios enviar mensajes y recibir respuestas generadas por un modelo de inteligencia artificial. 

En esta primera fase se construye la estructura semántica base y el diseño visual del chat, preparando el proyecto para futuras integraciones con APIs de modelos de lenguaje.

La aplicación está pensada bajo el enfoque Mobile First y con una arquitectura escalable para crecimiento futuro.

## Características Planeadas

- Interfaz de chat conversacional.
- Burbujas de mensaje diferenciadas (usuario / IA).
- Campo de entrada fijo en la parte inferior.
- Historial visual de conversación.
- Selector de modelo (fase futura).
- Integración con API externa (fase futura).
- Persistencia de conversaciones con Firebase (fase final).

## Instalación y Uso

1. Clonar el repositorio:

```bash
git clone https://github.com/TU-USUARIO/zenai.git
```

2. Ingresar al directorio:

```bash
cd zenai
```

3. Abrir el archivo `index.html` en el navegador.

No se requieren dependencias adicionales en esta fase.
